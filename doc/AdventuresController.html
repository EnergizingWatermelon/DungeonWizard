    create() click to toggle source

    Checks receives parameters and attempts to create a new Adventure and Map. If the adventure or map fail to save to the database for whatever reason, the user is redirected to the index.

    # File app/controllers/adventures_controller.rb, line 17
def create
    @adventure = Adventure.new(adventure_params)
    if (@adventure.save)
        @adventure.map = Map.new(map_params)
        if(@adventure.save)
            redirect_to @adventure
        else
            redirect_to '/adventures'
        end
    else
        redirect_to '/adventures'
    end
end
